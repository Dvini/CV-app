# 📄 Kreator CV – Nowoczesny Generator Życiorysów

![Kreator CV Preview](./preview.png)

**Kreator CV** to zaawansowana aplikacja webowa stworzona w celu ułatwienia procesu tworzenia profesjonalnych, estetycznych i skutecznych życiorysów. Dzięki intuicyjnemu edytorowi i podglądowi na żywo, użytkownik może w kilka minut przygotować dokument gotowy do wysłania rekruterowi.

---

## 🚀 Główne Funkcje

-   **✨ Edytor na żywo** – Wprowadzaj zmiany w danych i obserwuj, jak natychmiastowo zmienia się Twój dokument.
-   **🎨 Szeroka personalizacja** – Dostosuj kolory akcentów, typografię (integracja z Google Fonts) oraz odstępy między sekcjami.
-   **🌓 Motyw Dark/Light** – Przełączaj się między ciemnym a jasnym widokiem edytora dla większego komfortu pracy.
-   **🖼️ Elastyczny układ zdjęć** – Możliwość wyboru kształtu zdjęcia (koło, zaokrąglony kwadrat, prostokąt) oraz jego rozmiaru.
-   **📋 Rozbudowane sekcje** – Wsparcie dla doświadczenia, edukacji, projektów, kursów, języków, umiejętności i zainteresowań.
-   **⚖️ Klauzula RODO** – Łatwe dodawanie i formatowanie wymaganej klauzuli o ochronie danych osobowych.
-   **🖨️ Eksport do PDF** – Integracja z systemem drukowania przeglądarki zoptymalizowana pod format A4 (bez zbędnych marginesów i nagłówków).
-   **💾 Synchronizacja danych** – Możliwość eksportu i importu danych w formacie JSON, co ułatwia późniejszą edycję.

---

## 🛠️ Stos Technologiczny

Aplikacja została zbudowana z wykorzystaniem najnowocześniejszych narzędzi webowych:

-   **React 19** – Główny framework UI.
-   **Vite** – Błyskawiczne środowisko deweloperskie i system budowania.
-   **Lucide React** – Elegancki zestaw ikon.
-   **CSS3 (Vanilla)** – Wykorzystanie zmiennych CSS (custom properties) dla dynamicznego motywu i elastycznego układu.
-   **HTML5 Semantic Elements** – Dla lepszej dostępności i struktury dokumentu.
-   **AI Assisted** – Projekt rozwijany przy wsparciu zaawansowanych modeli językowych (Claude/Gemini) dla optymalizacji kodu i designu.

## ⚙️ Szybki Start

Aby uruchomić projekt lokalnie, wykonaj poniższe kroki:

1.  **Sklonuj repozytorium:**
    ```bash
    git clone https://github.com/TwojUser/app_cv.git
    cd app_cv
    ```

2.  **Zainstaluj zależności:**
    ```bash
    npm install
    ```

3.  **Uruchom serwer deweloperski:**
    ```bash
    npm run dev
    ```

Aplikacja będzie dostępna pod adresem `http://localhost:5173`.

---

## 📁 Struktura Projektu

-   `src/components/Editor` – Komponenty paneli bocznych do wprowadzania danych.
-   `src/components/Preview` – Silnik generujący podgląd dokumentu (paginacja, style CV).
-   `src/context` – Zarządzanie stanem aplikacji (CVContext).
-   `src/styles` – Globalne style, motywy kolorystyczne i definicje layoutu.
