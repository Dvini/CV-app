### [3. Pagination Fix: Keep Headings With Content]

**Implementation:**
* Currently, if a section's content (e.g. `cv-skills-list`) overflows and moves to the next page, its heading (`UMIEJĘTNOŚCI`) stays orphaned on the first page.
* We will tag headings with `data-keep-with-next="true"`.
* When the pagination engine pushes an element to the next page, it will check if the preceding element was its section heading. If so, it will push the heading instead, naturally dragging the content with it.
